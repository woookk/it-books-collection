## 디렉토리 구조
`it-books-collection` 저장소는 여러 개의 개별 저장소가 서브모듈로 구성되어 있습니다.
루트 디렉토리 하위에 개별 디렉토리가 존재하며, 각 디렉토리는 해당 책 저장소를 서브모듈로 추가합니다.

- [Tidy First](https://github.com/woookk/tidy-first)
